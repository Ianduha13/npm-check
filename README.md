# npm checker

A free-to-use site to check and analyze dependencies in your `package.json` file.

## Features
- 🚀 Quickly analyze your `package.json`
- 📦 Check for outdated dependencies

## How to Use

1. Upload your `package.json` or paste its content into the editor.
2. Click "Check Dependencies".
3. See the results and recommendations for updating your dependencies.
